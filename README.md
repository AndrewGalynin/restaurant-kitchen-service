# Restaurant Kitchen Service

A comprehensive Django web application designed to manage a restaurant's kitchen workflow. This system allows users to efficiently track dishes, dish categories, ingredients, and manage the kitchen staff (cooks).

## Features
* **Cook Management:** Register new cooks, update their years of experience, and assign them to specific dishes.
* **Menu Management:** Create, update, and delete dishes and dish types.
* **Ingredient Tracking:** Manage the inventory of ingredients required for various dishes.
* **Custom UI:** A clean, modern, and fully custom-styled user interface built from scratch without reliance on external CSS frameworks like Bootstrap.
* **Search Functionality:** Built-in search forms with pagination to easily find specific cooks, dishes, dish types, or ingredients.

## Technologies Used
* Python 3.14
* Django 6.0.5
* SQLite (Default database)
* HTML/CSS (Custom styling)
